[Jekyll]Tag기능 추가

05 Aug 2016
  • jekyll

블로그에 테마를 적용하고 태그만 작동하지 않아 그대로 두다가, 다른 블로그의 도움을 받아 기능을 추가했다.

원인은 github page에서 안전을 위해 플러그인을 제외시킨것. github page에서는 jekyll을 실행할 때 --safe옵션을 사용한다고 한다.

raw 폴더를 따로 만들고 페이지에 지킬 파일 전체를 올리는 대신 생성된 페이지만을 올려서 블로그가 정상적으로 열리도록 했다.

7번까지는 그대로 따르고, 추후 디렉토리 변경 등을 위해 8. 자동화에서 bash 파일을 변경하는 대신 raw 폴더에 상대경로를 사용한 스크립트를 추가했다.
